restore_domains -- Restore DataPower application domains from backup.

  • Synopsis
  • Parameters
  • Examples
  • Return Values
  • Status

Synopsis

Restore DataPower application domains from backup. This is not a secure restore.

Parameters

export_path (True, path, None)
Path to the domain export zip file.
overwrite_objects (False, bool, False)
Determines whether to overwrite objects that

exist in the current configuration during the import operation.

overwrite_files (False, bool, False)
Determines whether to overwrite files that exist

in the current configuration during the import operation.

rewrite_local_ip (False, bool, False)
Determines whether the local address bindings of services

in the import package are rewritten on import to their equivalent interfaces.

domains (False, list, None)
Indicates specific domains in the import package to import.

The dictionary can be composed of the following keys, name is minimum requirement. name: Specifies the name of the domain in the import package. import-domain: Determines whether to import the domain. Valid values are on or off. The default value is on. reset-domain: Determines whether to reset the domain before the import operation. Valid values are on or off. The default value is off.

files (False, list, None)
Indicates specific files in the import package to import.

The dictionary can be composed of the following keys, name is minimum requirement. name: The name of the file in the import package. overwrite: Overrides the setting of the OverwriteFiles parameter.

Examples

- name: Import the full_export
  br35ba56.datapower.restore_domains:
    overwrite_objects: yes
    overwrite_files: yes
    export_path: "{{full_export.export}}"

Return Values

request (always, dict, {'body': {'UnquiesceDP': {}}, 'method': 'POST', 'path': '/mgmt/actionqueue/default'})
The request that was sent to DataPower
response (on success, dict, {'_links': {'self': {'href': '/mgmt/actionqueue/default/pending/UnquiesceDP-20201231T105919Z-12'}}, 'status': 'completed'})
The response from DataPower
